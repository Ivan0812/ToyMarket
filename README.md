 --ToyMarket--

ToyMarket ist eine einfache Full-Stack Webanwendung für einen kleinen Online-Spielzeugmarkt.  
Das Projekt dient als Lern- und Portfolio-Projekt.

--Funktionen--

- Anzeige aller Produkte
- Featured Products Slider (Swiper)
- Produktdetailseite mit mehreren Bildern
- Warenkorb (Add / Remove)
- „Out of Stock“ Anzeige
- Responsive Design
- Backend API mit Express & MongoDB

-- Technologien --

-- Frontend --
- React
- React Router
- Swiper.js
- CSS / Tailwind CSS
- Vite

-- Backend --
- Node.js
- Express
- MongoDB (Mongoose)

-- Projektstruktur --

- `/frontend` – React Frontend
- `/backend` – Express API
- MongoDB Atlas als Datenbank

-- Installation -- 

```bash
-- Frontend --
cd frontend
npm install
npm run dev

-- Backend --
cd backend

npm run dev
